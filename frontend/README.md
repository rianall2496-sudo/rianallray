# 서울 코포레이션: 재벌의 탄생

## 💡 질문하신 내용에 대한 답변 (주가 패턴 및 서버 시간 흐름)

**Q1. 접속할 때마다 똑같은 패턴으로 주가가 시작하나요?**
**A1. 아닙니다!** 
현재 코드는 게임을 처음 시작할 때마다 **초기 주가에 -10% ~ +10%의 랜덤 변동성**을 부여하여 시작하도록 업데이트되었습니다. 또한, 주가가 움직이는 알고리즘 자체에 `Math.random()` 난수가 포함되어 있고, 1분마다 터지는 뉴스(호재/악재)도 무작위로 결정되므로 **게임을 할 때마다 주가 차트의 모양과 급등하는 회사가 완전히 다르게 전개**됩니다.

**Q2. 나중에 웹상에서 접속할 때 서버에서 보내주는 건 시간상으로 계속 흘러가고 있는 상태에서 접속하는 건가요?**
**A2. 네, 정확합니다!**
*   **현재 버전 (로컬 테스트용):** 현재는 백엔드 서버 없이 브라우저의 `LocalStorage`를 사용하기 때문에, 브라우저를 끄면 게임 시간도 일시 정지되고 다시 켜면 그 시점부터 이어서 시작됩니다.
*   **실제 서버 연동 시 (미래):** 나중에 PHP, Node.js, Spring 등의 **실제 백엔드 서버에 이 로직을 올리게 되면, 서버는 24시간 365일 내내 쉬지 않고 게임 루프(Tick)를 돌리게 됩니다.** 즉, 유저가 로그아웃하고 자는 동안에도 주식 시장은 계속 움직이고, 타 회원(NPC 또는 실제 다른 유저)들이 주식을 사고팔며, 배당금이 쌓이게 됩니다. 유저가 아침에 다시 로그인하면 밤새 변동된 실시간 데이터를 서버로부터 받아와서 화면에 뿌려주게 됩니다. (완벽한 실시간 MMO 타이쿤 방식)

## 🗺️ 부동산 지도 이미지 적용 방법 (필독)

부동산 탭의 배경에 올려주신 서울 지도 이미지를 적용하려면 다음 단계를 따라주세요.

1. 채팅창에 올려주신 **서울 지도 이미지를 다운로드**합니다.
2. 다운로드한 이미지의 파일 이름을 **`seoul_map.jpg`**로 변경합니다.
3. 이 이미지를 `index.html` 파일이 있는 **동일한 폴더**에 넣습니다.
4. 게임을 실행하고 부동산 탭에 들어가면 멋진 지도가 배경으로 깔려 있는 것을 확인할 수 있습니다!

## 아키텍처 안내 (Architecture Notice)

사용자님께서 `HTML, JS, PHP, MySQL` 스택으로의 변환을 요청하셨으나, 본 AI는 **React, TypeScript, Tailwind CSS 기반의 프론트엔드 싱글 페이지 애플리케이션(SPA) 코드만 생성하도록 엄격하게 설정**되어 있습니다. 

따라서 PHP 및 MySQL 코드를 직접 제공할 수 없는 점 양해 부탁드립니다. 대신, MySQL 데이터베이스의 역할을 대체하고 오프라인 환경에서도 데이터가 영구적으로 보존될 수 있도록 **브라우저의 LocalStorage를 활용한 로컬 DB 시스템**을 React 앱 내에 구현하여 적용했습니다.

## 실행 방법 (How to run)

이 프로젝트는 별도의 빌드 과정(Webpack, Vite, Create React App 등)이나 `npm install` 없이 브라우저에서 직접 실행되도록 설계되었습니다 (ES Modules 및 Import Map 사용).

1. 제공된 모든 파일(`index.html`, `index.tsx`, `App.tsx` 등)을 폴더 구조에 맞게 로컬 PC에 저장합니다.
2. 보안 정책(CORS)으로 인해 로컬 파일 시스템(`file://`)에서 직접 열면 ES Module이 로드되지 않을 수 있으므로, **로컬 웹 서버**를 통해 실행해야 합니다.

### 추천하는 로컬 서버 실행 방법:

**옵션 1: VS Code (가장 쉬움)**
1. VS Code에서 프로젝트 폴더를 엽니다.
2. 확장 프로그램(Extensions) 탭에서 **Live Server**를 검색하여 설치합니다.
3. `index.html` 파일을 열고 우클릭한 뒤 **"Open with Live Server"**를 클릭합니다.

**옵션 2: Python**
터미널(명령 프롬프트)을 열고 프로젝트 폴더로 이동한 뒤 아래 명령어를 입력합니다.
```bash
python -m http.server 8000
```
이후 브라우저를 열고 `http://localhost:8000` 으로 접속합니다.
